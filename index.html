What needs to be updated?
1
Config Required
Set project_name, budget, currency_symbol.
2
ResourcePlan Required
Role · Person · Rate · week dates (Mon YYYY-MM-DD). Names must match TimeEntries.
3
Phases Required
Phase name + start date. Names must substring-match TimeEntries task descriptions.
4
TimeEntries Required
Paste timesheet export. Column order: Date · Person · Task · Phase · Hours · Billable.
5
Jira_Dump Required
Connect "Jira Cloud for Sheets" add-on, hourly sync. Add Warn/Over Hours formula columns.
6
SLA_Lookup Optional
Story points → Warn/Over hours. Defaults provided — edit only if SLA thresholds differ.
7
Teamleaves Optional
One row per leave day: Resource · Leave Date · Type. Use Resource=ALL for holidays.
—
Spend_Analysis Auto
Formula-driven. Add ticket keys in col A and story points in col B only.
